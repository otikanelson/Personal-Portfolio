SCENORY - Storyboarding Tool

JavaFX MySQL MVC Graphics

A professional-grade personal story-boarding tool I created to help me make creative animations on youtube with a friend, featuring zoom/pan canvas, drag-and-drop panels, and MySQL persistence. Implements MVC with command pattern and glass-morphism UI design. Currently adding grid and export features (PDF, PNG) for comprehensive storyboarding assistance.

Key Features:

  • Zoom/pan canvas with smooth navigation
  • Drag-and-drop panel management
  • MySQL database persistence
  • Drawing tools: triangle, square, circle, rectangle
  • Line tool and comprehensive color palette
  • Text adder and eraser tool functionality

Technical Architecture:

Built with JavaFX and IntelliJ, serving as an entry point into professional app development. Implements MVC architecture with command pattern for robust functionality. Features glass-morphism UI design and provides users with a comprehensive drawing experience and professional storyboarding assistance.
