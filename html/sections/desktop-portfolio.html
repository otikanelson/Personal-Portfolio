SCENORY

Storyboarding & Art Application

JavaFX MySQL MVC Architecture IntelliJ

Professional storyboarding and art application built with JavaFX and MySQL. Features zoom/pan canvas, drag-and-drop panels, and MySQL persistence. Implements MVC architecture with command pattern and glass-morphism UI. Includes drawing tools for shapes, lines, text, color palette, and eraser. Currently adding grid and export features (PDF, PNG).

Key Features

  • Zoom/pan canvas with drag-and-drop panels
  • Comprehensive drawing tools (shapes, lines, text)
  • Color palette and eraser functionality
  • MySQL persistence for project management
  • MVC architecture with command pattern
  • Glass-morphism UI design
