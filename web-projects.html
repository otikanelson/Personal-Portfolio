DARB - Crowdfunding Platform

React.js Node.js MySQL Express.js

A comprehensive full-stack crowdfunding network platform built with React.js frontend and Node.js/Express backend with MySQL database, in order to conceptualize startup investing in nigeria via Crowdfunding processes. Features campaign creation and management, contribution management, secure investment processing, user authentication and authorization, payment integration, and a data visualization dashboard.

Key Features:

  • Campaign creation and management system
  • Secure investment processing and contributions
  • JWT authentication and user authorization
  • Payment gateway integration
  • Data visualization dashboard with analytics
  • Database optimization for scalable performance

Technical Implementation:

Frontend completed using React.js, Tailwind CSS, and Vite for optimal performance. Backend currently in progress featuring Node.js/Express with MySQL database. Implements RESTful APIs, JWT authentication, and focuses on modular design and scalability. Creates an intuitive user experience for both project founders and investors.

BIZZBRIDGE - Artisan Marketplace

React.js Node.js MongoDB Express.js

A comprehensive web platform designed for connecting Nigerian artisans with customers, facilitating service discovery and booking across 26+ craft categories. The platform features advanced search functionality, location-based filtering (Lagos LGAs), user authentication with role-based access, and a sophisticated booking management system.

Key Features:

  • 26+ craft categories with advanced search
  • Location-based filtering for Lagos LGAs
  • Role-based authentication (customers/artisans)
  • Sophisticated booking management system
  • Featured artisan management and portfolios

Architecture & Design:

Implementing responsive design with modern React architecture, RESTful API design, and MongoDB database optimization. The project includes featured artisan management, service portfolio displays, and a comprehensive user friendly dashboard for thorough service management. Focuses on creating seamless user experience for service discovery and booking workflows.
